# Logic
This document describes what abilities (without talismans) are required to beat various islands  
Beating the island is defined as collecting all gears, chests and missions(some exceptions). Eyetems are not a requirement.  
This is for the average person, so only basic intended routes to collect the items, with at most some tight movements. Get ready to side flip!

### General Info
- any ability = either dash, spin or bat
- all forms = dash, spin, snake and bat
- dash is also wall run
- i did forget that spin form can deal damage, so some of the dashes may be able to be done with spin drill in fights
- this will almost definitely need rebalanced, some of these are probably too hard for the casual player. we will cross that bridge when we come to it
